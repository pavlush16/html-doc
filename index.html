Предметне середовище: Предметним середовищем для цього проекту є розробка веб-сайту для персонального блогу. Для цього використовується сучасний стек технологій, включаючи HTML, CSS і JavaScript для розробки фронтенду. Для управління версіями та спільної роботи над проєктом використовується система контролю версій Git та хмарний сервіс GitHub. Для розробки бекенду використовується NodeJs та Express

Тема: СТРУКТУРА HTML-ДОКУМЕНТА. ВИБІР ПРЕДМЕТНОЇ ГАЛУЗІ. РОБОТА З ПОСИЛАННЯМИ, ТАБЛИЦЯМИ, ЗОБРАЖЕННЯМИ, СПИСКАМИ в HTML-ДОКУМЕНТІ.

Мета: придбати практичні навички роботи з HTML-документом, таблицями, зображеннями, посиланнями, списками, формами Створити шаблон звітного HTML-документом для відображення результатів роботи всіх лабораторних робіт.

GitHub: https://github.com/pavlush16/my-web-app/tree/main

// Класи дозволяють створювати об'єкти з властивостями та методами.
class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }

    greet() {
        return `Hello, my name is ${this.name} and I am ${this.age} years old.`;
    }
}

let person1 = new Person('Alice', 25);
console.log(person1.greet()); // Результат: Hello, my name is Alice and I am 25 years old.
